# 007 — PDF download endpoint

**Status**: [x] done

## Objective

Add `GET /api/articles/:id/pdf` that downloads and caches PDFs locally.

## Files

- `src/index.js` (update — add route)

## Spec

- Look up article by `id` or `doi`
- If PDF already cached at `data/pdfs/<doi>.pdf`, serve it directly
- Otherwise, fetch from `article.pdf_url` with 15s timeout
- Save to `data/pdfs/<doi>.pdf`, serve it
- Return 404 if article not found or no pdf_url

## Acceptance

- First request for an article downloads + caches the PDF
- Subsequent requests serve from local cache
- Missing PDF returns 404 gracefully
